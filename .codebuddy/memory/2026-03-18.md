# 2026-03-18 工作日志

## Hydration 错误修复

- **问题**: MarkdownRenderer 中 h2 渲染器使用可变计数器 `sectionCounter.current += 1`，SSR 和 CSR 分别执行产生不同编号，触发 Hydration mismatch
- **修复**: 改为 `useMemo` 预计算的 `buildH2IndexMap()` 映射表，通过正则扫描 Markdown 源文本预建 `标题文本 → 编号` 的 Map，纯函数无副作用

## 装饰元素多元化升级

对文章详情页的各种装饰线和视觉元素进行了多元化升级：

### 阅读进度条（顶部）
- 从单层 2px 改为 3px 多层结构：底层发光(blur) + 主渐变条 + shimmer 流光叠层
- 渐变色从纯金色改为 teal → 金色 → teal 的多色渐变
- 新增 CSS 动画：`progress-shimmer-move`（流光移动）、`dot-pulse`（末端粒子脉冲）

### 标题装饰分割线（header 底部）
- 从单一渐变线改为：左侧 teal 色短线 + 中央菱形几何装饰（3个旋转方块）+ 右侧渐变长线

### h2 章节标题
- 从简单的 `01` 编号前缀改为：左侧渐变竖线 + 底部圆点 + `Section 01` 标签 + hover 背景发光
- 奇偶章节交替使用 teal/gold 两种色彩

### h3 子标题
- 新增双圆点前缀装饰（teal + gold）

### hr 分隔符
- 从单一渐变线改为：左渐变线 + 中央几何装饰（圆点 + 旋转方块交替）+ 右渐变线

### 代码块终端圆点
- 从全灰改为 金色/teal/灰 三色

### Header 背景
- 新增第二条对角线（teal 色，方向相反）
- 新增 3 个浮动几何装饰点（使用 float-slow 动画）

### 文章底部分隔线
- 从简单 border-t 改为与 hr 风格一致的几何装饰分隔线
