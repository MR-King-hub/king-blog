## 渐变实现方案（重要）

Tailwind CSS v4 的 `@import "tailwindcss"` 可能导致自定义 CSS class 中的 `linear-gradient` 属性被处理异常，因此**所有渐变均使用内联 style 实现**，不走 CSS class。

- **文字渐变**: `linear-gradient(135deg, #DEC4A0 0%, #C9A87C 45%, #A8896A 100%)`（浅沙→沙铜→深铜）
  - 配合 `WebkitBackgroundClip: 'text'` + `WebkitTextFillColor: 'transparent'`
  - 应用于: Hero标题"技术阅读"、Navbar logo"Blogs"、ScoreMethod"评分方法"、Footer"优质技术内容"和"Blogs"
- **按钮/选中态渐变**: `linear-gradient(135deg, #C9A87C 0%, #A8896A 100%)`（沙铜→深铜）
  - hover态: `linear-gradient(135deg, #DEC4A0 0%, #C9A87C 50%, #A8896A 100%)`
  - 应用于: Hero CTA按钮、WeeklyPicks tab选中态、Navbar tab选中态、Footer订阅按钮、Navbar/Footer logo图标
